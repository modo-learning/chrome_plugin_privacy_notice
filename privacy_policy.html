Modo AI Coach — Privacy Policy

Last updated: March 4, 2026

What Data We Collect

Modo AI Coach records the following user interactions only while a capture session is actively running:

  • Click targets (element selectors and visible labels)
  • Text typed into non-sensitive form fields
  • Page navigations (URLs and page titles)

Sensitive inputs — including passwords, credit card numbers, security codes, and expiration dates — are never captured.

How Data Is Stored

All captured data is stored locally on your device using Chrome's built-in chrome.storage.local API. No data is transmitted to any external server, cloud service, or third party.

Data Sharing

Modo AI Coach does not share, sell, or transmit any user data. All data remains entirely on your device.

User Controls

  • Start/Stop: Recording only occurs when you explicitly start a capture session.
  • Site filtering: You can restrict recording to specific websites using the Tracked Sites settings.
  • Delete: You can delete any individual capture or all captures at any time from the extension popup.
  • Export: You can export captures as local Markdown files for your own use.
  • Uninstall: Removing the extension deletes all stored data.

Permissions

  • activeTab: Used to communicate with the current tab for recording interactions.
  • storage: Used to save captures locally on your device.
  • Content script on all URLs: Required so the extension can detect user interactions (clicks, typing, navigation) on any website you choose to record. The content script is inert on sites that are not being tracked, and recording is always user-initiated.

Changes to This Policy

If we make changes to this privacy policy, the updated version will be included with the extension update and the "Last updated" date will be revised.
